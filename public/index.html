Available Endpoints

GET /api

Get all destinations with optional query parameters for filtering.

Query Parameters:

  • continent - Filter by continent name
  • country - Filter by country name
  • is_open_to_public - Filter by open to public
Example: /api?continent=Europe&country=Italy&is_open_to_public=true

GET /api/continent/:continent

Get all destinations in a specific continent.

Example: /api/continent/Europe

GET /api/country/:country

Get all destinations in a specific country.

Example: /api/country/Italy

Response Format

All endpoints return JSON data in the following format:

{ "data": [...], "count": number, "message": "string" }

CORS

This API supports Cross-Origin Resource Sharing (CORS) and can be accessed from any domain.
